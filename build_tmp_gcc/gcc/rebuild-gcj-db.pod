=head1 NAME

rebuild-gcj-db - Merge the per-solib databases made by aot-compile into one system-wide database.

=head1 SYNOPSIS

rebuild-gcj-db

=head1 DESCRIPTION

C<rebuild-gcj-db> is a script that merges the per-solib databases made by
C<aot-compile> into one system-wide database so C<gij> can find the 
solibs.
